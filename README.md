# Rock, Paper, Scissors Game

This is a Rock, Paper, Scissors game built with HTML, CSS, and JavaScript. Challenge the computer in this classic hand game through a clean, fully responsive interface designed for both desktop and mobile play.

## Features

- Play Rock, Paper, Scissors against the computer.
- Human vs. AI: Play the classic game against a randomized computer opponent.
- Simple and intuitive user interface.
- Enjoy a seamless experience on any device size.
- Tracks your wins, losses, and ties.

## How to Play

1. Select Your Move: Click on the Rock, Paper, or Scissors button.
2. Computer's Turn: The computer will instantly make a random selection.
3. Result: The winner of the round is displayed immediately.
4. Repeat: Keep playing to see if you can outsmart the computer!
